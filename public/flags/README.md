# Drapeaux pour la Bibliothèque (langues)

Les **images des drapeaux** déposées ici sont utilisées dans le filtre par langue de la Bibliothèque (sélecteur à côté du titre). Les fichiers présents sont chargés automatiquement ; pour les langues sans image, l’emoji drapeau est affiché.

## Dimensions recommandées

- **Taille d’affichage dans l’interface :** 24 × 24 px
- **Fichiers à déposer :** **32 × 32 px** (ou **48 × 48 px** pour écrans haute densité)

Format conseillé : **PNG** (fond transparent si possible).

## Nom des fichiers

Un fichier par **code pays ISO 3166-1 alpha-2** en **minuscules** :

| Fichier   | Langue / pays |
|-----------|----------------|
| `fr.png`  | Français (France) |
| `gb.png`  | Anglais (Royaume-Uni) |
| `es.png`  | Espagnol |
| `de.png`  | Allemand |
| `it.png`  | Italien |
| `pt.png`  | Portugais |
| `nl.png`  | Néerlandais |
| `pl.png`  | Polonais |
| `ru.png`  | Russe |
| `jp.png`  | Japonais |
| `cn.png`  | Chinois |
| `sa.png`  | Arabe (Arabie saoudite) |
| `in.png`  | Hindi (Inde) |
| `kr.png`  | Coréen |
| `tr.png`  | Turc |
| `se.png`  | Suédois |
| `dk.png`  | Danois |
| `no.png`  | Norvégien |
| `fi.png`  | Finnois |
| `gr.png`  | Grec |
| …         | Autres selon besoin |

Si un fichier n’existe pas pour une langue, l’app affiche l’emoji drapeau (fallback). Après avoir ajouté ou modifié des fichiers ici, un simple rafraîchissement de la page Bibliothèque suffit pour les voir.
